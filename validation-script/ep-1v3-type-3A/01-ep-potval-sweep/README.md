## Potentiometer calibration

The HAMEG HM8112-3 was connected to the input of the energy profiler. The AUTO_RESISTANCE_TUNE parameter was disabled in the firmware. A sweep was performed over all potentiometer values to calibrate the energy profiler using a lookup table implemented in the firmware. The results of the calibrated energy profiler are presented below.
